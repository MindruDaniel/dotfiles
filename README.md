## Setup
- Dock: `plank`
- App Launcher: `rofi`
- File Manager: `ranger`
- Terminal Emulator: `rxvt-unicode`
- Text Editor: `geany`
- Music Player: `mpd`, `ncmpcpp` and `cava`
