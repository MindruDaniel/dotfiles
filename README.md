# DOTFILES

### About
Currently, i am using `openbox` and `windowchef` and this is the list of apps contained on my setup.

- Shell: `zsh`
- Dock: `plank`
- App Launcher: `rofi`
- Terminal Emulator: `rxvt-unicode`
- Text Editor: `vim`
- Music Player: `mpd` and `ncmpcpp`
- Visualizer: `cava`
